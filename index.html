About the Visualization

Interactive Slideshow This narrative visualization follows an interactive slideshow structure to guide users through various selected scenarios.

Scenes Three scenes have been included in this visualization using consistent template and layout. User's can navigate through the slideshow by clicking the arrows on the left/right sides of the screen, clicking the navigation dots on top, or using the left/right buttons on a keyboard.

Annotations Interesting trends are annotated in charts 1 and 2 using black font. These are meant to draw a user's attention to areas of the chart that are important to the overarching question behind the visualization.

Parameters Charts 2 and 3 include parameters that allow users to interact with the data via filtering. The scenes aim to answer a simple question and these parameters allow users to explore data in more detail.

Triggers Modifying the filters on charts 2 and 3 triggers changes to the parameters discussed above and updates the charts to reflect the event.
